かんたんボイスクローン for Qwen-TTS

最初にやること
1. `かんたんボイスクローン.app` を `Applications` へドラッグします
2. `Applications` から `かんたんボイスクローン.app` を開きます
3. アプリの最初の画面で `初回セットアップ` を押します
4. 準備が終わったら、そのままアプリ内の `起動` を押します

この DMG 版について
- Python はアプリ内に同梱しています。先に Python を入れておく必要はありません
- `qwen-tts` などの実行環境もアプリ側の内蔵ランタイムを使います
- 外部の Python や外部の `qwen-tts` を基本的には使いません
- ただし、モデルのキャッシュがすでに Mac にある場合は再利用されることがあります

注意
- 初回の音声生成時は、モデルのダウンロードや準備で時間がかかることがあります
- 動画から音声を取り出すには ffmpeg があると便利です
- ローカル文字起こしを使いたいときは、アプリ内の `ローカル文字起こしを追加` を押します
- 使い方をあとで見返したいときは、アプリ内の `READMEを開く` を押します

困ったとき
- 詳しい使い方は `README.md` を開いてください
