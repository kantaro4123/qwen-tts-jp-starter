かんたんボイスクローン for Qwen-TTS

最初にやること
1. このDMGの中にある `qwen-tts-jp-starter` フォルダごと、Macの好きな場所へコピーします
2. コピーしたフォルダを開きます
3. `かんたんボイスクローン.app` をダブルクリックします
4. 最初は `初回セットアップ` を選びます
5. セットアップ完了後、もう一度 `かんたんボイスクローン.app` を開いて `起動` を選びます

注意
- `.app` だけを単独で移動せず、必ず `qwen-tts-jp-starter` フォルダごと使ってください
- Python 3.9以上が先に入っている必要があります。おすすめは Python 3.12 です
- 動画から音声を取り出すには ffmpeg があると便利です
- ローカル文字起こしを使いたいときは `.app` のメニューから `ローカル文字起こしを追加` を選べます

困ったとき
- 詳しい使い方は `README.md` を開いてください
- 起動しない場合は `run.command` をダブルクリックしてみてください
