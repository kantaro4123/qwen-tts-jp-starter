# v0.5.0

配布者向けの公開導線を整理した版です。

## 追加したこと

- `1.7B` 標準同梱版を一発で作る `build_release_1_7b.sh` を追加
- GitHub Release まで一気に進める `release_1_7b.sh` を追加
- 公開時の最小チェックをまとめた `docs/release-checklist.md` を追加

## この版の意味

- 「初心者向けに配れる標準版はどれか」が分かりやすくなりました
- 配布者は `1.7B` を標準にした DMG を迷わず作れるようになりました
- Apple Developer 未加入でも、まずはそのまま配布を進めやすくなっています
