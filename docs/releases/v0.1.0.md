# v0.1.0

初心者向けの最初の公開版です。

## できること

- 日本語UIで Qwen-TTS のボイスクローンを試せます
- 参照音声だけでなく参照動画もアップロードできます
- 参照素材の切り出し、無音カット、軽い標準化に対応しています
- 日本語以外の複数言語で音声生成できます
- OpenAI API またはローカル faster-whisper で参照音声の自動文字起こしが使えます
- macOS 向け `.app` と `.dmg` 配布に対応しています

## 初心者向けの改善

- `.app` の中から `起動 / 初回セットアップ / 更新 / ローカル文字起こしを追加 / READMEを開く` を操作できます
- 起動後はブラウザではなく、アプリ内画面でそのまま使えます
- DMG に `はじめに.txt` を同梱し、最初の導線をわかりやすくしました

## 注意

- Apple Silicon の Mac を想定しています
- Python 3.9 以上が必要です。おすすめは Python 3.12 です
- 本人の声、または明確な許可がある声だけを使ってください
