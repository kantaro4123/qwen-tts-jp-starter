# v0.7.0

初心者向けの使い勝手と配布導線を大きく整理した更新です。

## 主な変更

- アプリ画面を `セットアップ` と `音声生成` で分離しました
- `設定` と `ログ` を別ウィンドウに分け、メイン画面から切り離しました
- ブラウザ版 UI を 2 カラム中心に作り直し、縦に長すぎる構成を整理しました
- `自動文字起こし` を目立つ位置に出し、初心者が流れを追いやすくしました
- `Use via API` などの英語 UI を隠し、日本語中心の見た目に揃えました
- GitHub Releases を見て更新があるときだけ、アプリ上部に `アップデート` ボタンが出るようにしました
- GitHub リポジトリへの導線をアプリ上部に追加しました
- 配布の主役を `DMG` から `ZIP 版アプリ` に切り替えました
- `はじめに.txt` をやめ、README に配布手順を一本化しました

## 配布について

- 標準のダウンロードは `qwen-tts-jp-starter-macos-app.zip` です
- `DMG` 版も併せて公開していますが、Gatekeeper の確認を減らしたい場合は `ZIP` 版をおすすめします
