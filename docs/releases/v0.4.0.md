# v0.4.0

standalone 配布をさらに進めた版です。

## 追加したこと

- `.app` 上部で `Qwen-TTS モデル` と `ローカルASR モデル` を選べるように変更
- 選んだモデル設定をアプリ側で保持し、初回セットアップや起動に反映するように調整
- ビルド時に任意の `Qwen-TTS` モデルを `.app` に同梱できる仕組みを追加
- `codesign` / `notarytool` を使った署名・公証スクリプトを追加

## この版の意味

- 初心者向けの DMG 版を保ちつつ、配布者向けにはより完成度の高い配布フローを用意しました
- モデル同梱版としてビルドすれば、初回生成時の待ち時間を減らせます
- Apple Developer の証明書がある環境なら、署名と公証の自動化にも進めます

## 補足

- 公証の実行自体には Apple Developer の認証情報が必要です
- このリリースではスクリプトと手順書を追加していますが、認証情報なしで実際の公証までは行っていません
