# v0.6.0

## 主な更新

- UI を整理して、アプリ版とブラウザ版の導線をわかりやすく調整
- 自動文字起こしを `faster-whisper` のローカル実行に一本化
- README を初心者向けに大きく整理
- 最新 UI に合わせてスクリーンショットを更新
- standalone ビルド時の安全チェックを追加
- `doctor.command` を追加して、状態確認を簡単に実行できるようにした
- DMG ビルドと配布導線を見直し

## 補足

- 文字起こしは、DMG 版ではアプリ内の `音声認識を追加（任意）`、ソース版では `./install_local_asr.command` で追加できます
- Apple Developer 未加入でも使えますが、初回起動時に macOS の警告が出る場合があります
