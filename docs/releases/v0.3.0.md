# v0.3.0

standalone `.app` 化を進めた版です。

## 追加したこと

- Python と主要な実行環境を `.app` 内に同梱
- 初回セットアップ時に、内蔵ランタイムを `Application Support` へ展開する方式に変更
- 起動後はブラウザではなく、アプリ内の WebView でそのまま使える構成を継続
- DMG を `Applications` へドラッグして入れる macOS らしい導線に変更
- 既存の外部 Python や外部 `qwen-tts` に依存しない配布形態に改善

## この版の意味

- DMG 版は、通常は Python を先に入れなくても使えます
- 既に Mac に Python や `qwen-tts` が入っていても、基本的にはアプリ内の同梱環境を使います
- そのため、既存環境を壊しにくく、初心者に配りやすいです

## まだ外に出るもの

- モデル本体は初回生成時にダウンロードされることがあります
- Hugging Face の既存キャッシュがある場合は再利用されることがあります

## 注意

- Apple Silicon の Mac を想定しています
- 本人の声、または明確な許可がある声だけを使ってください
