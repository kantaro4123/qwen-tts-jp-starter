# v0.2.0

初心者向けの導入体験をさらに改善した版です。

## 追加したこと

- `.app` の中に操作ボタンと表示画面を持つ macOS デスクトップアプリに変更
- `起動` 後はブラウザではなく、アプリ内の WebView でそのまま使えるように調整
- `READMEを開く` と `ローカル文字起こしを追加` もアプリ内導線から触れるように整理
- README 冒頭に DMG 導入イメージ GIF を追加
- GitHub Releases から DMG を直接配布しやすいようにリリーススクリプトを追加

## 配布の前提

- Apple Silicon の Mac を想定
- Python 3.9 以上が必要。おすすめは Python 3.12
- この段階では Python や `torch` を完全同梱した単体アプリではなく、同梱フォルダを使う軽量ラッパー構成

## 注意

- `.app` 単体ではなく `qwen-tts-jp-starter` フォルダごと使ってください
- 本人の声、または明確な許可がある声だけを使ってください
